# AI-Altermagnetic-MOFs
AI-accelerated Discovery of Altermagnetic Metal-Organic Frameworks via Interpretable Machine Learning

Welcome! 👋

This repo contains all the code and data for our paper on finding new altermagnetic MOFs using machine learning.

## What's inside?

Here's a quick tour of the folders:

### 📁 01_database
The full dataset from our paper (CSV format). Want all the raw DFT inputs/outputs for the MOF materials? Just email Yue Ran or Wei Jiang (see contact below). We're happy to share!

### 📁 02_alter_mat_all
All 15 altermagnetic MOFs we discovered. Each comes with VASP input and output files. Dig in!

### 📁 03_first_ML
The first-round ML models. Contains:
- Source code + data
- Training results
- `joblib` files if you want to reproduce everything

### 📁 04_second_classification_ML
Second-round classification models. Includes:
- Code + data
- Inputs & outputs
- SHAP analysis for interpretability

### 📁 05_second_regression_ML
Second-round regression models. Same setup as above:
- Code + data
- Inputs & outputs
- SHAP stuff to help you understand what's happening
